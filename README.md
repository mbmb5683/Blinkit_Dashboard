# Blinkit_Dashboard



# 📊 Sales Analysis Dashboard (Power BI)

## 🚀 Overview
This project showcases an interactive **Sales Analysis Dashboard** built using **Power BI**. The dashboard provides actionable insights into sales performance, item distribution, customer ratings, and outlet characteristics.

It helps businesses make data-driven decisions by visualizing key metrics in an intuitive way.

Analyzes Blinkit sales performance using Power BI dashboard
Shows total sales of $1.20M and 9K items sold
Displays average customer rating of 3.92
Helps understand customer behavior and product demand
Identifies top-performing and low-performing products
Supports data-driven decision-making
Improves operational efficiency and planning
Assists in creating effective business strategies for growth and profitability

---

## 📌 Key Features

- 📦 **Total Items Analysis** – Track number of items sold
- 💰 **Total Sales** – Overall revenue generated
- 📈 **Average Sales** – Average revenue per item
- ⭐ **Average Rating** – Customer satisfaction insights
- 🏪 **Outlet Analysis** – Performance based on outlet size, type, and establishment year
- 🥗 **Fat Content Analysis** – Compare low-fat vs regular items
- 🍎 **Item Type Breakdown** – Sales distribution by category

---

## 📊 Dashboard Insights

- **Soft Drinks** contribute significantly to sales
- **Tier 3 outlets** show strong performance
- **Medium-sized outlets** dominate in revenue generation
- Customer ratings remain consistently stable across categories
